# Finance App

A psychology-driven personal finance app built with Expo (React Native) and Next.js.

## Starting the app

The API server and Expo Metro bundler are tied to the terminal session. Closing the terminal stops both. To restart them:

**API server** (port 3001):
```bash
cd apps/api && pnpm dev
```

**Mobile app** (port 8081):
```bash
cd apps/mobile && pnpm dev
```

Open Expo Go on your phone and scan the QR code, or press `i` / `a` in the Metro terminal for iOS / Android simulator.
